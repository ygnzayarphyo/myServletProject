Registration API

URL: localhost:16522/SocialNetwork-war/RegisterServlet
Required: Content Type: application/x-www-form-urlencoded

Method Parameter Parameter Type Sample Description
POST txtUserName String(59) "Mg Mg" User Name
  txtEmail String(49) mgmg@gmail.com Email
  select String(3) +95 Country code for Phone Number
  txtPhoneNumber String(11) 09236457896 Mobile/Home phone number
  txtDOB String(10) 28/12/2000 Date of Birth (dd/MM/YYYY)
  txtPassword String(29) password User Password to login.
  returnType String(3) xml Default return data type.For now, only xml.

result:
status 11 means success
10 means already exist user
00 means System error


Login API

URL: localhost:16522/SocialNetwork-war/LoginServlet
Required: Content Type: application/x-www-form-urlencoded

Method Parameter Parameter Type Sample Description
POST txtEmail String(59) "Mg Mg" Email that register to system.
  txtPassword String(29) password User's password.
  returnType String(3) xml default return data type.For now, only xml.

result: status

  1. 11 means success
  2. 10 means User is not activate in this system.
  3. 01 means user not found.
  4. 00 means System error accountType (User is active or not).
  • 1 means active account and 0 means de-active account

Status API

URL: localhost:16522/SocialNetwork-war/list
Required: Content Type: application/x-www-form-urlencoded

Method Parameter Parameter Type Sample Description
POST title String(34) "Hello" Posted Status Title
status String(999) "About" Posted Status Messsage
date String(24) 2017 Both full date or only year or month is allowed
section String(10) status Default parameter "status" must be set.

result: <result> <row> <number>1</number> <title>Hello</title> <message>[something.]</message> <date>[2014-09-18 01:11:29.41]</date> </row> </result>

Active/ De-active account API

URL: localhost:16522/SocialNetwork-war/profile
Required: Content Type: application/x-www-form-urlencoded

Method Parameter Parameter Type Sample Description
